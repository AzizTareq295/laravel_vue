## Installing Process

<p style="font-size: 13px">After Clone or Download this project. Copy this project in your local server. First of all you have to generate laravel application key using command <b>" php artisan key:generate "</b> . Then using command <b>" composer update "</b> to updating all vendor. Use <b>" npm install "</b> to install vue and other dependencies. Import database from the project directory <b>db folder</b> . In the <b>.env</b> file you have to update <b>DB_USERNAME & DB_PASSWORD</b> as your local server username & password and restart your serve. After all of this, you have to run the command <b>" php artisan serve "</b> to run this project and also run this command <b>" npm run watch "</b>, command will continue running in your terminal and watch all relevant files for changes. </p>
